# Superstore Sales Dashboard

# Overview
This project is an **interactive Power BI dashboard** built using the Superstore dataset (Excel).  
It provides insights into sales performance through KPIs, trends, product categories, and region-wise analysis.

# Key Insights
- **KPI Cards**: Revenue, Orders, Average Order Value (AOV)
- **Line Chart**: Revenue Trend over time
- **Bar Chart**: Top-Selling Categories
- **Map Chart**: Region-wise Sales Comparison
- **Interactive Slicers**: Category, Region, Segment, Date

# Tools Used
- Power BI Desktop  
- Excel dataset (Superstore)  
- GitHub for project sharing

# How to Use
1. Download the `.pbix` file from this repository.  
2. Open it in **Power BI Desktop**.  
3. Use the slicers (Category, Region, Segment, Date) to interact with the dashboard.
